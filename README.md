# BRL Store – Full Stack E-commerce Web Application

## Overview

BRL Store is a full-stack e-commerce web application developed as part of the B204 App & Web Development module. The project demonstrates the implementation of a complete online shopping system including product browsing, cart management, and payment processing.

The application is designed with a premium dark UI and simulates a real-world logistics-based online store.

## Features

- Product listing and detail pages  
- Add to cart functionality  
- Dynamic cart total calculation  
- PayPal payment integration (sandbox)  
- Order storage in MongoDB  
- Session-based cart management  
- Search functionality  
- Premium responsive UI design  

---

## Technologies Used

### Front-End
- HTML (EJS templating)
- CSS (custom styling)
- JavaScript

### Back-End
- Node.js
- Express.js

### Database
- MongoDB Atlas

### Payment Integration
- PayPal Sandbox API

---

## System Architecture

The application follows a client-server architecture:

- Front-end (EJS + CSS) handles UI rendering  
- Express server handles routing and business logic  
- MongoDB Atlas stores products and orders  
- PayPal API processes payments
